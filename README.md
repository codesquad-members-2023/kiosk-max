# ☑ 커밋 메시지 컨벤션
    
```text
type #issue-number : subject
    
body
```
```text
feat #21 : 로그인
    
세션을 통한 로그인 기능을 개발
```

- `feat` : 새로운 기능
- `design`: CSS 와 같은 사용자 UI 변경
- `fix` : 버그 수정
- `docs` : 문서 수정 (README, .gitignore 등)
- `test` : 테스트 코드 추가
- `refactor` : 리팩토링
- `style` : 코드 의미에 영향을 주지 않는 변경사항
- `chore` : 빌드 부분 수정사항
